Listen mit HTML

Wie Listen schöner gestaltet werden können, sehen wir im CSS-Teil des Kurses.

Es gibt unterschiedliche Arten von Listen: Geordnete ( <ol>, für "ordered-list") und ungeordnete ( <ul>, für "unordered-list"). Der Unterschied besteht darin, dass geordnete Listen eine Nummerierung aufweisen.

Es gibt noch eine weitere Liste, die weniger in Gebrauch ist: die <dl>, description-list, die weder Aufzählungszeichen noch eine Nummerierung aufweist.
