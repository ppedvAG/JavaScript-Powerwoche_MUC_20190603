Tabelle in HTML

Wie die Tabelle schöner gestaltet werden kann, sehen wir im CSS-Teil des Kurses.

In den thead (Table-Head) kommen die Spaltenüberschriften. Die Tags dafür heißen <th>.

Der eigentliche Inhalt der Tabelle kommt in den tbody (table-body). Die Tags darin heißen <td> (für table-data). Sowohl <th> als auch <td> werden pro Zeile in ein weiteres Tag gepackt: <tr> (für table-row).
